# dragonsfire

A shitty version of [dragonfly](https://github.com/markevans/dragonfly), but for [crystal](http://crystal-lang.org)

Currently supported data stores:

  * File
  * (S3)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  dragonsfire:
    github: winks/dragonsfire
```
